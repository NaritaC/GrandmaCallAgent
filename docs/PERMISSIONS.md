# 权限说明

## Android 权限

`GrandmaBridge` 使用以下权限和服务：

| 权限/服务 | 用途 | 风险控制 |
| --- | --- | --- |
| `android.permission.INTERNET` | 为 V1 WebSocket 骨架预留 | V0 不启动网络 Runtime，且 Manifest 禁止明文流量；V1 必须使用受信任的 `wss://` 服务 |
| `AccessibilityService` | 识别微信来电并执行接听或用户主动触发的一键拨出 | 只处理 `com.tencent.mm`；每次点击前执行 SafetyGate，接听按钮使用精确标签匹配 |
| `NotificationListenerService` | 识别微信语音/视频来电通知 | 只读取通知标题和文本用于来电识别 |
| `TextToSpeech` | 播报安全提示或执行结果 | 不需要额外危险权限 |

## 用户授权

Android 系统不允许应用静默启用无障碍和通知监听。用户必须手动进入系统设置打开：

- 无障碍服务：`GrandmaBridge`
- 通知使用权：`GrandmaBridge`

## 数据最小化

第一阶段只上报：

- 设备心跳：设备 ID、型号、Android 版本、电量、授权状态。
- 来电事件：应用包名、联系人显示名、通话类型、来源。
- 执行结果：命令 ID、任务 ID、成功/失败、原因。

不上传聊天记录，不上传联系人通讯录，不上传音频、视频或屏幕截图。

## 明确禁止的动作

以下动作默认被 `SafetyGate` 拒绝：

- 支付、付款、转账、收款。
- 红包、银行卡、验证码相关操作。
- 删除消息、清空聊天记录、撤回消息。
- 主动发送消息、添加好友、修改账号资料。
- 任何非微信语音/视频来电接听的 UI 点击。

## 生产部署建议

- WebSocket 使用 `wss://`。
- 给每台设备配置独立设备令牌。
- 服务端日志脱敏并设置保留周期。
- 白名单变更必须有审计记录。
- 在老人手机上固定微信语言和版本，降低 UI 文案变化导致的误判。
